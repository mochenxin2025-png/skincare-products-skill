# skincare-products-skill

A Hermes Agent skill for skincare product analysis — ingredient scoring, evidence-based evaluation, and confidence-engine backed recommendations.

## Structure

- `SKILL.md` — Main skill definition and instructions
- `references/` — Product specs, scoring algorithms, ingredient ontology
- `assets/` — JSON schemas
- `platforms/` — Platform-specific outputs (e.g. Obsidian note templates)
- `scripts/` — Automation scripts
