# Ingredient Ontology Map

## INCI → Ontology Classification

Every ingredient maps to exactly ONE primary ontology category (M13).

Legend:
- `[A]` = Core Active
- `[FS]` = Formula Support
- `[SC]` = Skin Conditioning
- `[TM]` = Texture Modifier
- `[P]` = Preservative
- `[S]` = Solvent
- `[C]` = Chelator
- `[E]` = Emulsifier
- `[ST]` = Stabilizer
- `[FF]` = Film Former
- `[CO]` = Colorant
- `[FR]` = Fragrance
- `[B]` = Botanical
- `[M]` = Marketing
- `[R]` = Risk

Evidence levels: A (Meta-analysis), B (Clinical), C (In Vitro), D (Expert), E (Marketing)

## Usage Rules

1. Every ingredient is classified into exactly ONE primary ontology category before any efficacy analysis.
2. If an ingredient plausibly fits two categories, the more functionally significant one wins.
3. Botanical classification [B] is default for plant extracts unless purified active evidence exists (then reclassify to [A] or [SC]).
4. Risk classification [R] is additive — an ingredient can be [R] AND also belong to another category for scoring (e.g. Oxybenzone is both [A] as UV filter and [R] for endocrine concern).
5. Marketing [M] is reserved for ingredients with negligible or zero evidence of cosmetic function at typical formula concentrations.
