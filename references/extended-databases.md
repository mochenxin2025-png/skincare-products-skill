### 7.4 Silicone Safety & Environmental Notes

| Concern | Detail |
| --- | --- |
| D4 (Cyclotetrasiloxane) | Restricted in EU (REACH) for environmental persistence; avoid in rinse-off |
| D5 (Cyclopentasiloxane) | Under EU review; currently allowed; environmental persistence concern |
| D6 (Cyclohexasiloxane) | Similar concerns to D5; less volatile |
| Non-volatile dimethicones | No environmental concern; too large to bioaccumulate |
| Skin safety | All silicones are non-comedogenic, hypoallergenic, non-irritating |
| "Silicone-free" claim | Marketing; no scientific rationale for avoidance except personal preference |

### 7.5 Silicone Classification for Analysis

```
Category based on INCI name patterns:
  Cyclo[methicone/siloxane] → Volatile silicone
  Dimethicone (plain)       → Non-volatile linear silicone
  Dimethiconol             → High-viscosity terminal silanol
  [X] Dimethicone Crosspolymer → Elastomer
  [X] Methicone            → Functional/modified silicone
  Siloxysilicate           → Film former / resin
```

### 8.4 Polymer Selection Guide for Analysis

```
Want to assess a gel formulation?
  → Carbomer + TEA/NaOH/AMP neutralizer → traditional carbomer gel
  → Aristoflex AVC → pre-neutralized, salt-tolerant, elegant feel
  → Sepimax Zen → extreme electrolyte tolerance (Vitamin C serums)

Want to assess suspension quality?
  → Carbomer + Xanthan Gum → synergistic suspension
  → Microcrystalline Cellulose + Carbomer → excellent particle suspension

Want to assess waterproofing?
  → Acrylates/Octylacrylamide Copolymer → best water resistance
  → Trimethylsiloxysilicate + Acrylates Copolymer → silicone + organic hybrid film

Polymer system issues to flag:
  → Carbomer without neutralizer → won't thicken (formulation error)
  → Xanthan Gum at >1% → may feel sticky/tacky
  → Sodium Polyacrylate + low pH → loses viscosity
  → PVP in high-humidity → may feel tacky on skin
```

### 9.5 Botanical Assessment Rules

```
1. Distinguish extract vs. purified active:
   "Centella Asiatica Extract" → [B] (extract, unknown active concentration)
   "Madecassoside" → [A] (purified compound, known concentration)
   
2. Position matters:
   Extract in top 5 positions → may have functional concentration → [B] or [SC]
   Extract after fragrance → trace level → [M] regardless of evidence

3. Standardization matters:
   "Green Tea Extract (standardized to 90% EGCG)" → [A] (known active)
   "Green Tea Extract" (no standardization) → [B] (unknown potency)

4. Traditional use ≠ evidence:
   "Used for centuries in [region]" → Evidence D (expert/tradition)
   Only reclassify to C+ if modern studies exist

5. Overuse detection:
   If >5 botanical extracts after fragrance position → overuse → marketing penalty
```
